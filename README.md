# CSSのやつ

自分用のウェブデザインのためのCSS

## 特徴

- 近未来的なダークテーマデザイン

### グリッド
```html
<div class="container">
  <div class="grid">
    <div class="col-4">4列</div>
    <div class="col-8">8列</div>
  </div>
</div>
```

### UI要素
```html
<button class="btn btn-primary">ボタン</button>
<span class="tag">タグ</span>
<div class="alert">アラート</div>
<div class="card">カード</div>
<div class="progress"><div class="progress-bar"></div></div>
```

### アクセシビリティ
- セマンティックHTML5要素対応
- WAI-ARIA属性サポート
- キーボードナビゲーション
- スクリーンリーダー対応

### レスポンシブ
- モバイルファースト設計
- 768px以下に最適化
- Flexbox/Gridレイアウト
